# RNK-Web-Page
Mi proyecto es una página web con la temática del manga, Reincarnation no Kaben o Petals of Reincarnation, creada por el autor Mikihisa Konishi.
Decidí enfocar mi web de esta forma porque esta es una de mis historias favoritas y, dado que siempre me ha gustado explayarme bastante a la
hora de dar explicaciones (sobre todo con cosas que me gustan), pensé que lo mejor sería dedicar la web al trasfondo de toda la historia,
hablando de sus personajes, conceptos, etc.

Tecnologías utilizadas:
•	HTML
•	CSS
•	JavaScript
•	PHP
•	MySQL

Pasos de "instalación."
Para poder instalar la página web en tu dispositivo, primero tienes que ir al repositorio donde se encuentra subida. Para facilitar las cosas,
el enlace es el siguiente: https://github.com/Christie1611/RNK-Web-Page.

Tras eso, cuando veas la pantalla del repositorio, habrá un botón verde que dirá "<> Code", al darle click, en la parte baja habrá una opción
que te permitirá descargar toda la web en un archivo zip. Una vez descargado, descomprime el archivo.

Dentro de la carpeta habrá un archivo .sql, que es la base de datos con la que trabaja la web; necesitas exportarla a phpMyAdmin, en mi caso,
para tener acceso a dicho programa tuve que instalar xampp, por lo que es importante que tengas uno similar para poder seguir adelante.
Suponiendo que el ordenador ya posee phpMyAdmin, una vez estés dentro, ve a la opción que dice "importar", donde te permitirá importar la
base de datos de la web, una vez eso esté hecho, ya no debería de haber ningún fallo con la conexión a la base de datos.

Dado que la web trabaja con phpMyAdmin, es de vital importancia que toda la carpeta de la página se encuentre dentro del archivo htdocs en xampp. De esta forma, al entrar desde el localhost será posible acceder a la web.

Una vez dentro, puedes ir probando cualquier función que te llame la atención.
